Customer Shopping Behavior — EDA Project

Problem Statement

A leading retail company wants to better understand its customers' shopping behavior in order to improve sales, customer satisfaction, and long-term loyalty. The management team has noticed changes in purchasing patterns across demographics, product categories, and shipping preferences. They are particularly interested in uncovering which factors — such as discounts, reviews, seasons, subscriptions, or payment preferences — drive consumer decisions and repeat purchases.

You are tasked with analyzing the company's consumer behavior dataset to answer the following overarching business question:

"How can the company leverage consumer shopping data to identify trends, improve customer engagement, and optimize marketing and product strategies?"

Dataset

The dataset (customer_shopping_behavior__2_.csv) contains 3,900 customer records with 18 attributes, including customer demographics (age, gender, location), purchase details (item purchased, category, purchase amount, size, color, season), customer engagement signals (review rating, subscription status, previous purchases, frequency of purchases), and transaction details (shipping type, discount applied, promo code used, payment method).

Project Workflow


Data Cleaning & Preprocessing (Python / Pandas, in customerbehaviour.ipynb)

Inspected data structure and checked for missing values
Imputed missing Review Rating values using the category-wise median
Standardized column names (lowercase, underscores)
Renamed purchase_amount_(usd) to purchase_amount
Dropped promo_code_used after confirming it was identical to discount_applied



Feature Engineering

Created an age_group column (Young Adult, Adult, Middle-aged, Senior) using quartile binning on age
Converted frequency_of_purchases into a numeric purchase_frequency_days column for easier analysis



Loading into PostgreSQL

Loaded the cleaned dataset into a PostgreSQL database table (customer) using SQLAlchemy



SQL Analysis (customer_behaviour_sqlquery.sql)

Revenue generated by male vs. female customers
Top 5 products by average review rating
Average purchase amount: Standard vs. Express shipping
Spending comparison between subscribed and non-subscribed customers
Relationship between repeat buyers (5+ previous purchases) and subscription status
Revenue contribution by age group
